Introducción

El lenguaje HTML es un lenguaje de marcas de hipertexto que define el significado y la estructura del contenido web, pues es la base (esqueleto) de toda página web.

Es un lenguaje muy sencillo compuesto de elementos, que se pueden aplicar a piezas de texto para darles un significado diferente en un documento. ¡Veamos más de este lenguaje!

Etiquetas

Las etiquetas HTML son fragmentos o bloques de código que dan formato, funcionalidad y definen la estructura del contenido de las páginas web. Indican a un navegador la forma adecuada de la información contenida. Pueden influir en su presentación visual, como el tamaño, el color o el formato

Las etiquetas en su mayoría están compuestas de una etiqueta de apertura (por ejemplo, <p>) y una etiqueta de cierre (por ejemplo, <p>). El contenido a mostrar se coloca entre estas dos etiquetas.

Ejemplo:

  • <p>: Definen un párrafo.
  • <h1> a <h6>: Definen encabezados de diferentes niveles y tamaños.
  • <a>: Define un enlace (hipervínculo).
  • <img>: Inserta una imagen.
  • <ul> y <li>: Define listas no ordenadas y elementos de lista, respectivamente.

Enlaces

El elemento HTML <a> (de "anchor") se utiliza para crear hipervínculos. Se define con el atributo href, que indica la URL de destino.

Por otro lado, la etiqueta <link> se usa para enlazar recursos externos como hojas de estilo CSS.

Ejemplo de uso: <a href="https://www.ejemplo.com">Visita mi sitio web</a>

Listas

Las listas son elementos que posibilitan una mejor navegación en las páginas web, ya que permiten agrupar y reorganizar cierto contenido según se requiera. Para los archivos HTML, las listas se clasifican en ordenadas, desordenadas y descriptivas.

Existen varios tipos de listas, se mencionan algunas:

  1. Listas ordenadas <ol>
  2. Listas desordenadas <ul>
  3. Listas de descripción <dl>

Las listas ordenadas como su nombre lo indica se catalogan de forma ordinal, y en HTML se programan con el comando "ol". Se pueden ordenar ya sea con números o letras.

Por su parte las listas desordenadas usan puntos o cuadrados, y en HTML se programan con el comando "ul".

Algunas características de las listas pueden ser:

  • Organización semántica: ayudan a estructurar el contenido de una página web de manera lógica y significativa, facilitando la comprensión tanto para los usuarios como para los motores de búsqueda.
  • Diversidad de tipos: HTML ofrece diferentes tipos de listas (ordenadas, desordenadas y de definición) para adaptarse a diversas necesidades de presentación y contenido.
  • Accesibilidad: Las listas bien marcadas en HTML son accesibles para personas con discapacidad visual, ya que los lectores de pantalla pueden navegar por ellas fácilmente

Las listas de descripción tienen este estilo:

Estructura de pares término-definición
Las listas de descripción están diseñadas para presentar términos (etiquetados con <dt>) y sus respectivas definiciones o descripciones (etiquetadas con <dd>). Esto las hace ideales para crear glosarios, diccionarios o cualquier contenido donde se requiera asociar un nombre a una descripción.
Uso para glosarios y metadatos:
Las listas de descripción son comúnmente utilizadas para construir glosarios, donde cada término es una palabra o concepto, y su descripción es la definición correspondiente. También son útiles para mostrar metadatos, como pares clave-valor, donde la clave es el término y el valor es la información asociada.
Flexibilidad en la presentación:
Aunque la estructura básica es de pares, los elementos <dt> y <dd> pueden repetirse dentro de la misma lista de descripción. Por ejemplo, un término puede tener múltiples definiciones, o varios términos pueden compartir una misma definición. Esto permite adaptar la lista a diferentes necesidades de presentación de contenido.
